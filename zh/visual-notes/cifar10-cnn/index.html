02 数据来源与结构

数据入口：理解数据集、预处理和批次加载怎样进入训练过程。

对应正文

读代码时补充理解的问题
开头的常量有什么用？
  • 代码开头的 CIFAR10_MEAN 和 CIFAR10_STD 不是模型训练出来的参数，而是 CIFAR-10 数据集 RGB 三个通道的均值和标准差。它们会被传给 transforms.Normalize，用于把图像张量做标准化：
  • 先用 ToTensor() 把像素值转换到 [0, 1] 附近。
  • 再用 (像素值 - 均值) / 标准差 调整每个颜色通道的分布。
固定随机种子是为了什么？
  • set_seed 的作用是尽量固定实验中的随机性。训练过程里有很多随机来源，例如模型初始权重、训练集打乱顺序、随机裁剪、随机水平翻转、随机擦除和 Dropout。
  • 固定随机种子不等于提高模型能力，它主要是为了让实验结果更可比较。如果后续只改一个函数或一个超参数，固定随机性可以减少“这次准确率变化只是随机波动”的干扰。

03 模型与特征图

模型结构：CNN 先提取中间特征，再把特征送入分类层。

对应正文

什么叫“提取特征”？
  • “提取特征”可以理解为：模型把原始像素逐步转换成更适合分类的表示。
  • 原始输入只是在描述每个位置的 RGB 数值；经过卷积层之后，中间特征图可能逐渐对应边缘、颜色块、纹理、局部形状，再到更接近类别判断的抽象模式。这些模式不是人工写死的，而是卷积核参数在训练中学出来的。
  • 所以，卷积层不是直接回答“这是不是飞机/汽车/鸟”，而是先把图像变成一组中间表示，最后分类层再根据这些表示输出 10 个类别分数。
ConvBlock(in_channels, out_channels, dropout) 里的参数是什么意思？
  • in_channels 表示输入特征图数量。第一层输入是 RGB 图像，所以是 3；后面的层接收上一层输出的特征图，所以可能是 64、128 或 256。
  • out_channels 表示当前卷积块要输出多少张新的特征图。可以把它理解为当前层要学习多少种新的特征探测方式。
  • dropout 表示训练时随机丢弃一部分通道的比例，用来减少模型对某些局部特征的过度依赖，从而缓解过拟合。
一个输出通道和卷积核是什么关系？
  • 一开始容易误以为“一个通道就是一个 3x3 卷积核”。更准确的理解是：
  • 一个输出通道对应一组卷积核参数。
  • 如果输入有 in_channels 张特征图，那么一个输出通道就需要 in_channels 个 3x3 小卷积核。
卷积和池化分别怎样改变特征图？
  • 在这次作业的模型里，普通卷积主要改变通道数量，不改变特征图宽高。原因是卷积层使用了 kernel_size=3、stride=1 和 padding=1。这种设置会让输出宽高和输入宽高保持一致。
  • 宽高仍然是 32x32，但通道数从 3 变成 64。
  • 池化层则主要负责缩小宽高。MaxPool2d(kernel_size=2) 默认步长为 2，所以会把宽高减半，但不改变通道数：

04 训练循环

训练循环：从前向计算到反向传播，再到参数更新和测试评估。

对应正文

几个基础问题
训练是在调整参数，参数存储在哪里？
  • 在 PyTorch 中，模型参数存储在模型对象内部。更具体地说，参数属于模型中的各个层，例如卷积层、线性层和 BatchNorm 层。
  • model = Cifar10CNN().to(device)
  • 这个 model 对象内部就包含了所有可训练参数。卷积层中的卷积核权重、线性层中的权重和偏置、BatchNorm 中的缩放和平移参数，都会被 PyTorch 记录为可训练参数。
CNN 通常由哪些部分组成？
  • 一个典型的卷积神经网络通常由多个卷积层和池化层组成。卷积层负责从局部区域提取特征，例如边缘、纹理、颜色组合和局部形状；激活函数负责引入非线性；池化层负责缩小特征图尺寸，减少计算量，并让模型对小范围平移更鲁棒。

  • 在这次作业中，模型的主体就是多个卷积功能块和池化层组成的特征提取器，最后接一个分类层输出 CIFAR-10 的 10 个类别。

05 概念与结果

结果与概念：把训练结果和容易混淆的基础概念放在一起复盘。

对应正文

一个 PyTorch 训练脚本通常是什么结构？
  • 一个典型的 PyTorch 训练脚本通常包含以下几部分：
  • 准备训练集、测试集和 DataLoader
  • 定义损失函数、优化器和学习率调度器
AdamW 优化器起什么作用？
  • 优化器的作用是根据梯度更新模型参数。最基础的梯度下降可以理解为：参数沿着让 loss 下降的方向移动一小步。
  • Adam 会在普通梯度下降的基础上记录梯度的历史信息，为不同参数自适应地调整更新幅度。这样通常更容易训练，也不需要像普通 SGD 那样细致地手动调学习率。
  • AdamW 是 Adam 的改进版本，它更合理地处理权重衰减。权重衰减可以理解为一种正则化方法，会抑制参数变得过大，从而减轻过拟合。
BatchNorm 是什么？
  • BatchNorm 是 Batch Normalization，中文通常译为批量归一化。它一般放在卷积层之后、激活函数之前。
  • 它的作用是让中间特征的数值分布更加稳定。对 CNN 来说，BatchNorm2d 会按通道统计特征的均值和方差，并对特征进行归一化。同时，它也有自己的可训练参数，用来学习缩放和平移。

训练环境与结果记录
  • 这次训练使用本机的 NVIDIA GPU 完成。GPU 本身负责执行大量矩阵运算和卷积运算，而 CUDA 可以理解为连接深度学习软件和 NVIDIA GPU 的计算平台。PyTorch 通过 CUDA 版本的 torch 把张量和模型参数放到 GPU 上，使前向传播、反向传播和参数更新都能在 GPU 上执行。
  • 从使用者角度看，CUDA 的作用主要体现在两点：一是让 PyTorch 能识别并调用 NVIDIA GPU；二是提供适合深度学习计算的底层加速能力。代码中把模型和数据移动到 cuda:0 后，后续计算就会尽量在 GPU 上完成。

  • BatchNorm 是 Batch Normalization，中文通常译为批量归一化。它一般放在卷积层之后、激活函数之前。
  • 它的作用是让中间特征的数值分布更加稳定。对 CNN 来说，BatchNorm2d 会按通道统计特征的均值和方差，并对特征进行归一化。同时，它也有自己的可训练参数，用来学习缩放和平移。

训练环境与结果记录

  • 这次训练使用本机的 NVIDIA GPU 完成。GPU 本身负责执行大量矩阵运算和卷积运算，而 CUDA 可以理解为连接深度学习软件和 NVIDIA GPU 的计算平台。PyTorch 通过 CUDA 版本的 torch 把张量和模型参数放到 GPU 上，使前向传播、反向传播和参数更新都能在 GPU 上执行。
  • 从使用者角度看，CUDA 的作用主要体现在两点：一是让 PyTorch 能识别并调用 NVIDIA GPU；二是提供适合深度学习计算的底层加速能力。代码中把模型和数据移动到 cuda:0 后，后续计算就会尽量在 GPU 上完成。
